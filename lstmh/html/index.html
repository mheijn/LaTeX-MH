1 Introduction

This package contains my language definitions of HTML5, JavaScript, and Cascading Style Sheets for the package listings

2 Install

latex listmh.ins
bash install.sh

This also will install (in latex/mh) or update lstlang0.sty and lstconf.cfg to link lstmh.sty and lstmhlang.sty to listings.

3 Usage

\lstinputlisting[ language=<lng>]index.html
Where <lng> can be
JavaScript,JS: Java Script CSS, Cascading Style Sheets HTML5: HTML5 WEB: HTML5, JS and CSS. 

\lstinputlisting[ language=WEB ]{index.html}
